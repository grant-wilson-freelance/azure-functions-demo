Azure Static Web App Demo

https://github.com/grant-wilson-freelance/azure-functions-demo

This repo uses Azure Static Web Apps for hosting this frontend built with Angular 15 (/angular-app), and Azure Functions (/api) for the REST API.

This can be run locally using npm start for each project, and also deployed to Azure using GitHub Actions. The GitHub action configuration file is in /.github/workspace/azure-static-web-apps-zealous-moss-023c75610.yml
